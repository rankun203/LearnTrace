 成都职业技术学院

    *软件技术专业

        本专业是国家骨干高职建设院校重点专业，拥有一支省级教学团队。
        本专业与成都知名软件企业一起， 共同培养具有软件开发能力的高素质高技能实用人才。
        本专业设置了3个专业方向，即Java开发、.Net开发、移动应用开发。毕业生可在企事业单位、
        各类软件公司及信息服务平台运营公司从事软件开发与维护、软件测试、业务流程管理等工作。
